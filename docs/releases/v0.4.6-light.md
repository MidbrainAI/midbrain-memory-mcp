# v0.4.6 Light Notes

v0.4.6 makes MidBrain installer setup cleaner for multi-client users and
replaces unconditional debug logs with bounded, leveled logging.

Global installs now write one shared key to `~/.config/midbrain/.midbrain-key`
by default. Interactive multi-client installs ask whether clients should share
that key; users can still choose distinct per-client keys, and existing
distinct per-client keys are preserved. Non-interactive installs always stay on
the shared-key path and no longer rewrite per-client key files.

Logging now goes through `shared/logger.mjs` with `makeLogger()`, `logFile()`,
and `logDir()`. Hooks and plugins use platform log directories, level
thresholds, and one-file rotation. Debug-level request detail stays suppressed
unless `MIDBRAIN_LOG_LEVEL=debug` is set.

The release includes installer, logger, hook, API, and OpenCode runtime tests,
plus full project check and package dry-run coverage. Breaking changes: none.
