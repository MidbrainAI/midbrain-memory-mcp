# midbrain-memory-mcp v0.4.1

v0.4.1 is a patch release for the OpenCode and NanoClaw setup paths introduced
in `0.4.0`.

Upgrade or install:

```bash
npx -y midbrain-memory-mcp@latest install
```

## Highlights

- Fixes the OpenCode `Plugin export is not a function` loader failure by
  exporting only callable plugin functions.
- Hardens OpenCode plugin repair by comparing installed plugin and bundle
  contents, not only a marker file.
- Adds stable NanoClaw hook commands through the published package:

  ```text
  npx -y midbrain-memory-mcp@latest hook claude user
  npx -y midbrain-memory-mcp@latest hook claude assistant
  ```

- Updates NanoClaw docs and the `/add-midbrain` skill to avoid pinned
  `/pnpm/.../midbrain-memory-mcp@<version>/...` hook paths.

## Important NanoClaw Note

Existing NanoClaw groups already pinned to `midbrain-memory-mcp@0.3.2` or to a
package-store hook path are not automatically repaired by this release. They
need one migration to replace the old MCP and hook config with `@latest`
commands. After that, they should update normally on future cold starts.

## Codex

No Codex runtime fix was present in the verified `v0.4.0..HEAD` diff. Codex
behavior remains unchanged from `0.4.0`.

Breaking changes: None.
