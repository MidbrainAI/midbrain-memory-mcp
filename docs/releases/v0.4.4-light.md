# v0.4.4 Light Notes

v0.4.4 makes first-time MidBrain setup much simpler.

Users can now sign in at memory.midbrain.ai, run
`npx midbrain-memory-mcp install`, and let the installer complete browser-based
auth, create or select a memory agent, create an API key, write key files, and
patch supported client configs.

Manual key setup is still available with `--no-login`. That flag now correctly
skips browser/device auth and stays on the manual fallback path.

The README Quick Start now reflects the new sign-in, install, restart flow, and
the release adds device-auth and installer-boundary tests for the FTUX path.
