# v0.4.1 TLDR

v0.4.1 is a patch release for OpenCode and NanoClaw.

- Fixes OpenCode `Plugin export is not a function`.
- Makes OpenCode freshness repair compare installed plugin and bundle contents.
- Adds `npx -y midbrain-memory-mcp@latest hook claude user|assistant`.
- Makes new NanoClaw hook configs use:

  ```text
  npx -y midbrain-memory-mcp@latest hook claude user
  npx -y midbrain-memory-mcp@latest hook claude assistant
  ```

- Existing pinned NanoClaw groups still need one migration from old `0.3.2` or
  `/pnpm/...@<version>/...` hook paths to `@latest`.
- Codex behavior is unchanged from `0.4.0`.
- Breaking changes: None.

Upgrade:

```bash
npx -y midbrain-memory-mcp@latest install
```

Version check after publish:

```bash
npx -y midbrain-memory-mcp@latest --version
# 0.4.1
```
